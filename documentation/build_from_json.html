magicExcel Documentation

build_from_json

This tag will parse query-format JSON to generate an Excel document

This uses JSON query format. (JSON = { COLUMNS = [], DATA = [] })

Syntax

  <cfimport taglib="/path/to/magicExcel" prefix="magicExcel">
  
  <magicExcel:build_from_json
    json="#json_string#"
    file="/path/to/excel/document.xls"
  />

Tag Options

Attribute Req/Opt Default Description

custom_theme

Optional

 

Using a custom theme definition will override the theme option.

To create your own theme definition, you will need to create a struct in the following format (items in bold are required):

  custom_theme = {
    name = "spooky_theme_1",
    definition = {
      heading = {
        bold = "true",
        color = "orange",
        fgcolor = "black",
        fillpattern = "solid_foreground"
      },
      data = {
        color = "dark_orange",
        fillpattern = "nofill"
      }
    }
  }

For theme definition options, please refer to the ColdFusion 9 documentation for SpreadsheetFormatCell().

file

Required

 

Expanded path to the file you wish to generate.

json

Required

 

Query-Formatted JSON string or variable that contains one.

theme

Optional

black_on_white

Use a built-in, colored theme to your excel document.

  • blueberry
  • lemon
  • strawberry
  • cherry
  • lime
  • orange
  • licorice
  • white_on_black

variable

Optional

build_from_json_result

Since there is no browser output, this variable will be set to determine if the file was generated.

Possible values returned are:

  • OK
  • FAIL
